Detailed Description

Classe per la gestione della comunicazione con un server FTP.

Author
Grandieri Andrea g.and.nosp@m.reus.nosp@m.02@gm.nosp@m.ail..nosp@m.com
Barsotti Mirko bmirk.nosp@m.o150.nosp@m.5@gma.nosp@m.il.c.nosp@m.om
Brambilla Mirko bramb.nosp@m.obil.nosp@m.la01@.nosp@m.gmai.nosp@m.l.com

La seguente classe fornisce tutti i metodi necessari a stabilire una connessione e ad relazionarsi con un server FTP.

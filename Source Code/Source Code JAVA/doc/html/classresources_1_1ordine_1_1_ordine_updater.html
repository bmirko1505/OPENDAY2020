void resources.ordine.OrdineUpdater.run ( )

Codice eseguito dal thread.

Il seguente metodo controlla l'eventuale presenza di nuove ordinazioni nel server in attesa di essere scaricate e caricate localmente nella lista InterfacciaPrincipaleGalataio.jListListaGustiVendita. Se presenti, le scarica, le carica nella lista InterfacciaPrincipaleGalataio.jListListaGustiVendita e infine le elimina dal server.

Detailed Description

Classe per mantenere "corrente" e visualizzare correttamente l'ora nelle interfacce del software.

Author
Grandieri Andrea g.and.nosp@m.reus.nosp@m.02@gm.nosp@m.ail..nosp@m.com
Barsotti Mirko bmirk.nosp@m.o150.nosp@m.5@gma.nosp@m.il.c.nosp@m.om
Brambilla Mirko bramb.nosp@m.obil.nosp@m.la01@.nosp@m.gmai.nosp@m.l.com

La seguente classe rappresenta un thread daemon per mantenere sincronizzata l'ora visualizzata nelle interfacce del software con l'ora del sistema operativo. Inoltre, permette di visualizzare in modo corretto quest'ultima.

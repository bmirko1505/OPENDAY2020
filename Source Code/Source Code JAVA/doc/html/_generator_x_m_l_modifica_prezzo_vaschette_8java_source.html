243  // - source -> documento "virtuale": il documento "virtuale" che fa da supporto all'albero DOM. In dettaglio ho bisogno solo dell'albero DOM
244  // contenuto/"appoggiato" su questo documento "virtuale"
245  // - target -> il file XML "vero e proprio": il file XML "vero e proprio" nel quale scrivere l'albero DOM convertito.

Detailed Description

Classe per formattare e visualizzare correttamente un elemento della lista InterfacciaPrincipaleGelataio.jListOrdini.

La seguente classe effettua una formattazione delle informazioni da visualizzare nella lista InterfacciaPrincipaleGelataio.jListOrdini di un suo elemento. Questo permette una visualizzazione di quest'ultime informazioni più ordinata e accattivante.

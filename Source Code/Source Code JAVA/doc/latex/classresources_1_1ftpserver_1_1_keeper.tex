\doxysubsection{Detailed Description}
Classe per la gestione del servizio keep alive. 

\begin{DoxyAuthor}{Author}
Grandieri Andrea \href{mailto:g.andreus02@gmail.com}{\texttt{ g.\+andreus02@gmail.\+com}} 

Barsotti Mirko \href{mailto:bmirko1505@gmail.com}{\texttt{ bmirko1505@gmail.\+com}} 

Brambilla Mirko \href{mailto:brambobilla01@gmail.com}{\texttt{ brambobilla01@gmail.\+com}}
\end{DoxyAuthor}
La seguente classe rappresenta un thread daemon per fornire il servizio keep alive. Il servizio keep alive permettere di mantenere una connessione con il server anche quando non avviene uno scambio di informazioni client-\/server per un periodo di tempo prolungato. Dunque, il servizio keep alive garantisce la connessione anche in sessioni idle.
